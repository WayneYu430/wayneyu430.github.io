    • 仔细阅读Bomb.c，会发现我们是通过不同的输入，共有6个phase，然后引爆炸弹的。直接断点到37（main行）

    disas phase_1

    call a function, and we can see it test the value in the %eax(which is lower 32bit in %rax, the return value). mov $0x402400,%esi means we mov a value in the %esi, which is the second arguments.

    So what is the phase_1() before? Let’s return to the main()

    the read_line() function mov %rax into the %rdi, where the rax% is the return value, %rdi is the first argument.

    read_line函数会将读入字符串地址存放在rdi 和rsi中，strings_not_equal函数会使用edi和esi中的值当做两个字符址，并且判断他们是否相等，相等返回0
