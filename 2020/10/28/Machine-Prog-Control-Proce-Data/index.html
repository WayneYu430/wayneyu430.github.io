Condition Codes

Condition Means
CF Carry Flag (进位，overflow, unsigned)
SF Sign Flag
ZF Zero Flag
OF Over Flag (signed)
  • lea 不产生这4个码

  1. 两种产生方式

    1. Implicitly set （as side effect）

    2. Explicit set

      • testq b, a like a & b
      • test one parameter

    3. Explicit read by Set

      根据条件码置Reg的低位，且 可读

      movzblzero rest of %rax

      规定：

      在x86中，若结果为32位，则置高位32位bit为0

      若结果为short，char等，只影响16bit，8bit
